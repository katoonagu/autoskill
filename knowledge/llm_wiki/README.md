# LLM Wiki

This folder is the shared long-term memory layer for all agents.

Current scope:

- human-readable knowledge pages
- structured summaries and decisions
- no embeddings
- no vector DB
- no RAG yet

- `brands/`
- `bloggers/`
- `contacts/`
- `campaigns/`
- `conversations/`
- `evidence/`
- `decisions/`
