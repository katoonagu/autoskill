# Instagram Brand Search Module

This folder contains the Instagram sponsorship / brand-discovery module.

## Main Files

- `models.py` — data models
- `state.py` — resumable crawler state
- `job.yaml` — job config
- `plan.md` — implementation plan
- `runners/run.py` — module runner scaffold

## Inputs

- `inputs/instagram_brand_search/blogger_profiles.txt`

## Entry Point

- wrapper: `scripts/run_instagram_brand_search.py`
